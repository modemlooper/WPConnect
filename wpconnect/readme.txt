== Description ==
Allows users to log into your site with a WordPress.com user account.

== Installation ==
= Automatic Installation =

1. From inside your WordPress administration panel, visit 'Plugins -> Add New'
2. Search for `WPConnect` and find this plugin in the results
3. Click 'Install'
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Enter WordPress.com application settings on WPConnect admin page

= Manual Installation =

1. Upload 'wpconnect' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter WordPress.com application settings on WPConnect admin page
